
  # animarka

  This is a code bundle for animarka (formerly College Grade System App).

  ## Running the code

  Run `npm install` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build with PWA assets.

  ## PWA

  The app is now configured as a Progressive Web App using `vite-plugin-pwa`.

  It includes:

  - A generated service worker for offline caching.
  - A web manifest for installability.
  - App icons in the `public` folder.
